# GitHub 101 - challenge

## System Requirements
- git
- GitHub account

## Challenge
- Fork devopsloft/training repo
- Clone it
- Checkout to a branch
- Implement a meaningful change to this challenge
- Submit a PR
- Pass a code review
- Verify your PR is merged
- Update your local and origin repo with the merged changes
